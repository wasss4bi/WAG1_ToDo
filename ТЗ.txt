Тудушка бамбам
1. Регистрация
    - Никнейм
    - Почта
    - Пароль
2. Вход
    - Почта
    - Пароль
3. Папки задач
    - Создание (Кнопка "создать" - Ввести название - создать)
    - Удаление (три точки - удалить)
    - Редактирование названия (три точки - редактировать)
4. Задачи
    - Заголовок
    - Тип задачи (Запланированное количество чего-либо, разовое дело)
    - Дата постановки задачи
    - Дата окончания выполнения задачи
    - Важность (A - красный, B - оранжевый, C - жёлтый, D - зелёный)
    - При наведении появляется карандашик, открывающий правое окно, в котором будет содержаться вся информация об этой задаче
